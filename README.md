Professional Resume Builder

An interactive Resume Builder web app built using HTML, CSS, and Vanilla JavaScript.
It allows users to create a professional resume in real-time, preview it instantly, and download it as a PDF.

🚀 Features

🖊️ Real-time Resume Editing – Update personal details, experience, education, and skills with instant live preview.

📌 Dynamic Sections – Add or remove multiple experiences and education entries.

🎨 Modern UI/UX – Glassmorphism, gradients, shadows, and smooth hover effects.

🏷️ Skills Tagging – Enter comma-separated skills, automatically styled as tags.

📑 Download as PDF – One-click PDF download using print functionality.

📱 Responsive Design – Works seamlessly on desktop and mobile.

🛠️ Tech Stack

HTML5 – Structure

CSS3 – Styling, glassmorphism, responsive layout

JavaScript (Vanilla JS) – Live preview updates, dynamic section handling, PDF download

🎯 How to Run

Clone or download this repository:

git clone https://github.com/your-username/resume-builder.git


Open the project folder.

Run index.html in your browser.

Fill in details → Preview updates instantly → Download as PDF.
